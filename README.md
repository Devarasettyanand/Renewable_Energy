# Renewable_Energy
This mini-project demonstrates an end-to-end cloud data pipeline using AWS S3 → Snowflake → Tableau to analyze renewable energy usage trends. It simulates how real companies build cost dashboards, consumption insights, and cloud-usage reporting systems.
